gsap.to("CSS Element Selector", {

```javascript
gsap.from("CSS Element Selector", {
    // CSS Properties(camelCase)  & GSAP Properties
})

```


## Some Properties

```javascript
gsap.to("CSS Selector", {
    x: 1000,    // Translate in X-Axis by 1000px
    y: 500,
    duration: 2,
    delay: 1,
    rotate: 360,
    scale:  1.2,
    borderRadius: "1rem 8rem 1rem 8rem",
    repeat: 3,
    yoyo: true
})
```
